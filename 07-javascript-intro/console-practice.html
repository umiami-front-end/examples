Console Practice

Setup

Open the JavaScript console in either Google Chrome or Firefox.

Google Chrome: View → Developer → JavaScript Console

Firefox: Tools → Web Developer → Web Console

Instructions

Type each line of code, one at a time, in the Console. Hit enter to execute the statement. The process will look a little like this:

This practice designed to get you familiar with the console and basic JavaScript syntax. You are not expected to know the parts of any of these statements. Just practice entering this data.

Try being curious. See what a statement does when you change the data. Sometimes the best way to learn code is to break it!

If you are every curious about how a bit of JavaScript works, testing it in the console is a good way to experiment and find out!

Practice

console.log("Hello");

Writes "Hello" to the console.

document.write("Hello");

Writes "Hello" to the browser. Refresh the page when you finish this step.

window.alert("Hello");

Displays "Hello" in an alert box.

1 + 1;

3 - 2;

4 * 5;

Displays each result of the expression to the console.

Math.sqrt(25);

Returns the square root of 25.

Math.PI;

Returns the value of Pi (3.14...).

"hello".toUpperCase();

Transform "hello" to uppercase letters.

"car".replace("r", "t");

Replace the "r" in "car" with a "t".

var cats = 2;

The variable cats stores a value of 2.

cats;

Returns the value stored in cats. (It should return 2.)

clear();

Clear the console.
